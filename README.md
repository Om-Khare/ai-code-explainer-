# AI Code Explainer

A simple web application that explains code using AI.

## Features
- Explains code in simple terms
- Step-by-step breakdown
- Time complexity analysis

## Tech Stack
- HTML
- CSS
- JavaScript
- Gemini API

## How to Run
1. Clone the repository
2. Open index.html using Live Server
3. Add your own API key in script.js

## Note
API key is not included for security reasons.
